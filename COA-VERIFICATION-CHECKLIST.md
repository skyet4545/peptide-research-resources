# COA Verification Checklist for Research Peptides

A Certificate of Analysis (COA) is the primary quality indicator for research peptides. This checklist helps researchers evaluate vendor COA credibility.

## ✅ COA Verification Checklist

### Identity & Source
- [ ] COA issued by **third-party laboratory** (not in-house/vendor's own lab)
- [ ] Lab name is visible and searchable (verify it exists as a real accredited facility)
- [ ] Lab has ISO 17025 accreditation or equivalent
- [ ] COA has a unique batch/lot number matching the product label
- [ ] COA date is recent (within 12 months of purchase is ideal)

### Purity Testing
- [ ] **HPLC purity ≥98%** (High-Performance Liquid Chromatography)
- [ ] HPLC chromatogram is included (not just a number)
- [ ] Single dominant peak visible (no suspicious secondary peaks >2%)
- [ ] Mass Spectrometry (MS) confirms correct molecular weight

### Identity Confirmation
- [ ] Molecular formula matches known peptide sequence
- [ ] MW (molecular weight) matches published literature ±0.5 Da
- [ ] Amino acid sequence verified via MS/MS fragmentation (top-tier vendors)

### Safety Testing
- [ ] **Endotoxin/LAL test** result (<1 EU/mg preferred)
- [ ] Sterility testing (USP <71> or equivalent)
- [ ] Residual solvent testing (particularly for acetonitrile from HPLC purification)
- [ ] Heavy metals screening (optional but preferred)

### Document Integrity
- [ ] COA is downloadable as PDF (not just a screenshot)
- [ ] Lab contact information visible on document
- [ ] You can cross-reference batch number with the lab directly

## 🚩 Red Flags

- COA from "our in-house lab" with no third-party verification
- Purity listed as 99.9%+ without HPLC data (suspiciously perfect)
- No date on the COA
- COA shows different batch number than product
- Lab name that doesn't appear in any accreditation database
